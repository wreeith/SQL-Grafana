# SQL-Grafana
Структура проекта: 
1. createrightdata.py 
Python-скрипт, который создает, обрабатывает или подготавливает "правильные" (right) данные.
Что он может делать:
Генерировать тестовые или эталонные данные.
Преобразовывать сырые данные в нужный формат (например, очищать, фильтровать, агрегировать).
Подготавливать данные для использования в других скриптах или системах.

2. dataright.csv (Ac)
ВеCSV-файл, содержащий правильные данные.

3. knn.py (Cr)
knn Python-скрипт (Cr)
Реализовывать алгоритм k-средних для кластеризации данных.
Загружать данные
Проводить кластеризацию, визуализировать результаты или оценивать качество кластеризации.
Сохранять результаты (например, метки кластеров, центроиды) в файлы.
